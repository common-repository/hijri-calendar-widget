Wordpress plugin for historically accurate Hijri conversions.

== Description ==

This plugin's aim is to help with calendar conversions from gregorian dates to hijri dates and vice-versa.

Some of the features:

*   instant conversion of Gregorian Dates to Hijri Dates
*	Light weight widget

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-plugin-hijri-calendar.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
